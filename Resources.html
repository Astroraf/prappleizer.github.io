Resources

Here you will find a variety of resources that are useful when learning Python. Some have been created by me, others I link to with attribution.

Cheat Sheet: Unix Commands

Far from a full description (a more full version is in the textbook, and in various other books), this is a quick cheat sheet for common commands used when working in Unix-type systems. That is, when you use a terminal on a Linux system (or indeed a Mac).

Cheat Sheet: Vim Commands

While most people use more fully-featured and comprehensive text editors (or indeed, full blown IDEs), vim (and it's even more stripped down cousin vi) are the text editors that are guaranteed to be native on any code-related operating system you will be working on (probably). And if you are trying to edit code through an SSH connection or the like, you need to be able to edit your code from the terminal. This guide has all the basic commands needed to open, edit, and close your documents via this interface. It has a bit of a steep learning curve, but once you have it down, it's a very efficient way of making quick changes to code.

Guide: Python For Data Analysis

This guide put together by Nathanael A. Fortune (Smith College) is easy to follow, and has some good advice on simple python data analysis tasks, in particular plotting with error bars, etc. I recommend giving it a look!
