Difficulty Level: Easy

  • Guided tutorial (static html)

Unguided Tutorial

  • Import the neon spectrum text file and plot the spectrum.
  • Write a peak-finding algorithm to locate the peaks of the spectrum, specifying a threshold (up to you, maybe 20,000+).
  • Iterate over those peaks and calculate the center of light (same formula as center of mass) for each peak, using the FWHM (full width at half maximum) as the region over which you calculate the centroid.
  • Plot the centroids as vertical dashes over the spectrum, and then plot the residual between each centroid and peak.

Tutorial: Basic Data I/O and binned plotting with SDSS

Difficulty Level: Intermediate

Downloads / Dependencies:

  • Numpy, matplotlib, pyfits (or astropy.io.fits)
  • The SDSS DR7 Catalog
  • Unguided Tutorial (see below)
  • Guided tutorial(jupyter notebook)
  • Guided tutorial (static html)
  • Tutorial Solution (jupyter notebook)
  • Tutorial Solution (static html)

SDSS (Sloan Digital Sky Survey) is a large sample of nearby galaxies, and in addition to being a valuable tool for a variety of scientific goals (the survey has produced over 5,800 scientific papers, which in turn have been cited a staggering 245,000 times), it also provides a useful practice set on which to exercise our python chops.

Problem Statement

We want to first learn how to read in the fits files (data tables) storing the SDSS catalogs for 3 key measurements: Mass, Star Formation Rate (SFR), and metallicity. Once we have successfully accessed the data, we can start to look at one of the fundamental relations in galactic astronomy- Mass/Metalicity. Instructions:

  • Load the three catalog fits files into python and access the rec.array tables.
  • Filter the data to only include those for which entries could be properly measured, using the "AVG" column. For mass, take only those with values not equal to -1, for SFR take those with values >-99, and for metallicity take those with values >-99.9.
  • Make plots for mass vs metallicity, one of the fundamental correlations in extragalactic astronomy. Use a 2d Histogram with 300 bins (the LogNorm() function in matplotlib will be useful for colorbar purposes).
  • Make the same plot for Metallicity as a function of SFR and SFR as a function of mass.
  • You should notice all three exhibit positive trends. In order to determine which trends are legitimate and which are just representing the other trends, create three sets of plots in which you take slices (say, ten bins of equal width across the sample) of mass, metallicity, and sfr, and compare the other two properties. For one of them, the relation will dissapear. What is your conclusion then about which relations are real and which are derivatives?

Tutorial: A Beginner's Gude to Markov-Chain Monte Carlo (MCMC)

Difficulty Level: Easy (guided), Hard (unguided)

Downloads / Dependencies:

  • Guided tutorial (static html)

  • Solution (static html)

Unguided Tutorial (Hard)

  • Import the Ice core data and plot temperature (column 5) as a function of age (column 3).
  • Write a model function that produces a model that is a composite of three Sines, with input parameters as the three amplitudes, periods, and single (added) offset value T0.
  • Write a Likeness function that returns a value sum((y-y_model)^2/y_unc^2)
  • Write a prior function that checks if a set of parameters is within reasonable values for each - return 0 if yes, and -inf if not.
  • Write an lnp function that takes a list of parameters, runs it through the prior checking function, and if it returns 0, calculate a likelihood and return that.
  • Look up the documentation for emcee on how to set up an MCMC using these functions, and fit the ice core data. Overplot some models from your posterior distribution over your data.
